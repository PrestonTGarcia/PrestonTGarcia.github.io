This GUI is a visualization for projectile motion. The project helped me learn about GUI design, and how to use GUIs to best visualize physics disciplines.

This projectile motion visualization is implemented using [Tkinter](https://docs.python.org/3/library/tkinter.html) and [Matplotlib](https://matplotlib.org/), which are both Python libraries. When I first made this project, it took me two weeks as I added a few more buttons; however, I revisited the project recently since my laptop with the project broke down. In 2 days, I was able to create the GUI that visualizes the trajectory of a projectile given initial x, y, and speed values, with adjustable time points and angles.

In this project I gained experience of Python libraries, and how to think of efficient ways to visualize data using Python.
